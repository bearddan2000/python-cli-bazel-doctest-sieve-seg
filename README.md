# python-cli-bazel-doctest-sieve-seg

## Description
Given a number find all primes.
POC for doctest with bazel.

## Tech stack
- python
- bazel
  - custom script

## Docker stack
- l.gcr.io/google/bazel:latest

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
